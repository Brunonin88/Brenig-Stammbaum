GENEALOGIA BRENIG

Installationsanleitung für Android & Desktop

METHODE 1: PWA – Direkt auf Android installieren

Die empfohlene Methode – keine App-Store-Freigabe nötig. Funktioniert offline nach dem ersten Laden.

📱 ANDROID (Chrome) Empfohlen

1
Die index.html auf einen Webserver laden (z.B. Netlify Drop – einfach den Ordner per Drag & Drop hochladen, kostenlos)
2
Die URL in Chrome auf deinem Android-Gerät öffnen
3
Oben rechts das ⋮ Menü öffnen → „Zum Startbildschirm hinzufügen"
4
Auf „Installieren" tippen → App erscheint wie eine native App im Launcher
5
Beim nächsten Start: vollständig offline nutzbar (D3.js + Schriften werden beim Erstaufruf gecacht)

🖥️ DESKTOP (Chrome/Edge)

1
Gleiche URL im Browser öffnen
2
In der Adressleiste das ⊞ oder 💾 Symbol klicken → „Genealogia Brenig installieren"

METHODE 2: Echte APK-Datei erstellen Fortgeschritten

Wenn du eine echte .apk Datei möchtest (z.B. zum direkten Teilen):

🔧 Via PWABuilder (kostenlos, online)

1
App zuerst per Netlify veröffentlichen (wie oben)
2
pwabuilder.com öffnen → URL eingeben
3
Android auswählen → „Build App" → .apk herunterladen
4
Auf Android-Gerät: Einstellungen → Sicherheit → „Unbekannte Quellen erlauben" → APK installieren

📦 Via WebViewGold oder DioApp (kostenpflichtig)

Diese Tools verpacken deine HTML-Seite in eine vollständige native Android-App mit Play-Store-Unterstützung.

INHALT DIESES PAKETS

index.html – Die vollständige interaktive App (D3.js Stammbaum)
manifest.json – PWA-Konfiguration (Icon, Name, Vollbild-Modus)
sw.js – Service Worker für Offline-Nutzung
icon.svg / icon-192.svg / icon-512.svg – App-Icons
💡 Tipp Netlify Drop: Gehe zu app.netlify.com/drop → ziehe den gesamten Ordner brenig-app in das Browserfenster → du erhältst sofort eine URL wie brenig-stammbaum.netlify.app. Kostenlos, kein Account nötig.
